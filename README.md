# Instrument Management System

Aplicação desenvolvida em Microsoft Power Apps para gerenciamento e rastreabilidade de instrumentos.

## Objetivo

Facilitar o controle de instrumentos por meio de uma interface intuitiva, permitindo consultas rápidas, organização das informações e rastreabilidade dos ativos.

## Funcionalidades

- Cadastro de instrumentos
- Consulta de instrumentos
- Rastreabilidade
- Organização dos dados
- Interface desenvolvida em Microsoft Power Apps

## Tecnologias

- Microsoft Power Apps
- Power Fx

## Aprendizados

Durante o desenvolvimento deste projeto foram aplicados conceitos de:

- Organização de processos
- Desenvolvimento Low-Code
- Lógica de programação
- Modelagem de telas
- Experiência do usuário (UX)
